QE 安装

两种安装方法（Ubuntu：quantum-espresso）

1. GNU(compile)

sudo apt-get install 
- gfortran （Compiler）
- libblas-dev（Math library）
- liblapack-dev（Math library）
- libfftw3-dev （Math library）
- openmpi-bin libopenmpi-dev （Parallel library）
- gedit （Text editor）
- make （linux make）
Download QE（wget + http）
tar -zxvf qe-raleasepack.tar.gz
./configure 
make all 
Environment configuration of QE & openmpi 
whereis : find path of QE & openmpi
gedit ~/.bashrc
export PATH=$PATH:/home/lei/qe/bin
export PATH=$PATH:/usr/lib/x86_64-linux-gnu/openmpi/bin
export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/usr/lib/x86_64-linux-gnu/openmpi/lib
source ~/.bashrc
Then find some examples online to start your first project of QE （successfully）

2. intel(compile)

Download   oneMKL + oneHPC （wget + http ）
bash l_HPCKit_p_2021.3.0.3219_offline.sh 
bash l_onemkl_p_2021.3.0.3219_offline.sh
1.oneMKL （Math library）
2.oneMPI（Parallel library）included in oneHPC 
3.compiler （ifort、icc、icpc）included in oneHPC
Environment configuration of  oneMKL + oneHPC
gedit ~/.bashrc 
source ~/intel/oneapi/setvars.sh intel64 --force
source ~/.bashrc
Download QE（wget + http）
tar -zxvf qe-raleasepack.tar.gz
./configure --prefix=/home/lei/qe CC=icc CXX=icpc FC=ifort
make all 
Environment configuration of  QE
gedit ~/.bashrc 
export PATH=$PATH:/home/lei/qe/bin source ~/.bashrc
source ~/.bashrc
Then find some examples online to start your first project of QE（successfully）
