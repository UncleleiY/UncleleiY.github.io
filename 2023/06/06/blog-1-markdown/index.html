Markdown 基础语法

Markdown 简介

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，使用该语法编写的文档可以导出 HTML 、Word、PDF、Epub 等多种格式的文档。

# 一级标题

### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

Markdown 区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号

> 注意区分它们之间的差别

10.Markdown 表格

Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行；

|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |

单元格 单元格

表格对齐方式；

-: 设置内容和标题栏居右对齐。
:- 设置内容和标题栏居左对齐。
:-: 设置内容和标题栏居中对齐。

11.Markdown 转义字符

Markdown 支持以下这些符号前面加上反斜杠来帮助插入普通的符号

\   反斜线
`   反引号
*   星号
_   下划线
{}  花括号
[]  方括号
()  小括号
#   井字号
+   加号
-   减号
.   英文句点
!   感叹号

示例：

**文本加粗** 
\*\* 正常显示星号 \*\*

文本加粗
** 正常显示星号 **
