hexo 配置问题

本地hexo无响应

输入 hexo g && hexo s

Usage: hexo <command>
''''''

周而复始一直没办法执行命令

更改文件位置之后出现新的问题:

ERROR Cannot find module 'hexo' from '/home/uncle_yang/blog'
ERROR Local hexo loading failed in ~/blog
ERROR Try running: 'rm -rf node_modules && npm install --force'

输入 rm -rf node_modules && npm install –force

再执行hexo命令就解决问题了

新增解决方法

需要进入blog目录下，而不要在具体某个文件夹下去执行hexo的命令

复盘（1）

文件之类的东西务必时刻记住备份！备份！备份！！！

复盘（2）
